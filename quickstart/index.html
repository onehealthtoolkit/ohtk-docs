  • Getting Started

OHTK can be installed locally as a self-hosted project, or it can be hosted via the OHTK Cloud Service.

In most cases, you'll want to start with installing the software locally in this order:

  1. ohtk-api
  2. ohtk-ms
  3. ohtk-mobile
